## ğŸš€ **Getting Started**

### Prerequisites
- Modern web browser
- Basic image editing software (for profile photo)

### Setup Instructions
1. Clone or download the repository
2. Place your professional photo in `assets/profile.jpg` (420x420px recommended)
3. Add project screenshots to `assets/images/` folder
4. Update personal information in `index.html`
5. Deploy to your preferred hosting service

### Customization
- Edit CSS variables in `style.css` for theme colors
- Modify project data in `script.js` (getProjectsData function)
- Update social links in `index.html`
- Adjust animations in CSS and JavaScript
