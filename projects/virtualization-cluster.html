Key Features

High Availability

Automatic VM restart on healthy hosts during hardware failures with configurable admission control.

Fault Tolerance

Zero-downtime protection for mission-critical VMs through continuous replication to secondary hosts.

Shared Storage

iSCSI-based shared datastore enabling vMotion, HA, and centralized VM storage management.

Resource Management

DRS-ready cluster with resource pools for workload balancing and performance optimization.

Implementation Highlights

  • Deployed two ESXi 8.0 hosts with nested virtualization for lab environment
  • Configured vCenter Server Appliance (VCSA) for centralized cluster management
  • Established iSCSI shared storage with multipathing for redundancy
  • Enabled vSphere HA with host monitoring and VM restart priority settings
  • Configured Fault Tolerance for critical VMs requiring zero-downtime
  • Created distributed virtual switches for consistent network configuration
  • Implemented resource pools to allocate compute resources by workload type
